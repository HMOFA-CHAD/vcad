You've created a model—now let's export it! vcad supports multiple export formats, but STL is the most common for 3D printing.

## STL Export

The simplest way to export is with `.write_stl()`:

```rust
use vcad::centered_cube;

let cube = centered_cube("box", 30.0, 30.0, 30.0);
cube.write_stl("cube.stl")?;
```

This creates a binary STL file that any slicer can read.

<Callout type="info" title="Binary vs ASCII">
vcad always writes binary STL files. They're smaller and faster to process than ASCII STL. All modern slicers support both.
</Callout>

## Getting STL Bytes

For programmatic use, you can get the STL as bytes:

```rust
let stl_bytes: Vec<u8> = cube.to_stl();
// Write to custom location, send over network, etc.
```

## Available Export Formats

vcad supports multiple formats for different use cases:

| Format | Extension | Use Case |
|--------|-----------|----------|
| STL | `.stl` | 3D printing, CNC |
| glTF/GLB | `.glb` | Web viewers, rendering |
| USD | `.usda` | Isaac Sim, Omniverse |
| DXF | `.dxf` | Laser cutting (2D) |
| STEP | `.step` | CAD interchange |

## glTF Export with Materials

For visualization with materials, export to glTF:

```rust
use vcad::{Part, export::export_glb};
use vcad::export::MaterialDef;

let part = Part::cube("box", 30.0, 30.0, 30.0);

let material = MaterialDef {
    name: "Aluminum".into(),
    color: [0.9, 0.9, 0.92],
    metallic: 0.95,
    roughness: 0.3,
    density: None,
    description: None,
};

export_glb(&part, &material, "cube.glb")?;
```

## Multi-Part Scenes

For assemblies with multiple materials, use `Scene`:

```rust
use vcad::{Part, Scene};
use vcad::export::{Materials, export_scene_glb};

let materials = Materials::parse(r#"
    [materials.steel]
    color = [0.7, 0.7, 0.72]
    metallic = 0.9
    roughness = 0.4

    [materials.rubber]
    color = [0.1, 0.1, 0.1]
    metallic = 0.0
    roughness = 0.9
"#)?;

let mut scene = Scene::new("assembly");
scene.add(Part::cube("frame", 100.0, 50.0, 30.0), "steel");
scene.add(Part::cylinder("wheel", 20.0, 10.0, 32), "rubber");

export_scene_glb(&scene, &materials, "assembly.glb")?;
```

## DXF for Laser Cutting

For 2D profiles (laser cutting, waterjet), use DXF:

```rust
use vcad::export::DxfDocument;

let mut doc = DxfDocument::new();
doc.add_rectangle(100.0, 60.0, 0.0, 0.0);  // Outer profile
doc.add_circle(0.0, 0.0, 15.0);             // Center hole
doc.export("bracket.dxf")?;
```

<Callout type="tip" title="Bend Lines">
For sheet metal, add bend lines on a separate layer:

```rust
doc.add_bend_line(-50.0, 20.0, 50.0, 20.0);
```

These appear on the BEND layer in the DXF file.
</Callout>

## Inspecting Your Model

Before exporting, you can inspect your model's properties:

```rust
let part = centered_cube("box", 30.0, 30.0, 30.0);

println!("Volume: {:.2} mm³", part.volume());
println!("Surface area: {:.2} mm²", part.surface_area());
println!("Triangles: {}", part.num_triangles());

let (min, max) = part.bounding_box();
println!("Bounding box: {:?} to {:?}", min, max);
```

## Verification Checklist

Before sending to a printer:

1. **Check volume** — Does it make sense for your model?
2. **Check triangle count** — Very high counts may slow slicing
3. **Check bounding box** — Is it the expected size?
4. **Open in slicer** — Visual inspection for holes or errors

<Callout type="info" title="Watertight guarantee">
vcad uses the Manifold kernel, which guarantees watertight output. If your inputs are valid, your exports will be too—no mesh healing required.
</Callout>

## Complete Example

Here's a complete example that creates and exports a mounting plate:

```rust
use vcad::{centered_cube, centered_cylinder, Part};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Create plate
    let plate = centered_cube("plate", 100.0, 60.0, 5.0);

    // Create hole pattern
    let hole = centered_cylinder("hole", 3.0, 10.0, 32);
    let holes = hole
        .linear_pattern(80.0, 0.0, 0.0, 2)
        .linear_pattern(0.0, 40.0, 0.0, 2)
        .translate(-40.0, -20.0, 0.0);

    // Combine and export
    let part = plate - holes;

    println!("Volume: {:.2} mm³", part.volume());
    part.write_stl("plate.stl")?;

    Ok(())
}
```
