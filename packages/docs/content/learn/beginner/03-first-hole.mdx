Creating a hole is one of the most common operations in CAD. In vcad, you create holes using **boolean difference**â€”subtracting one shape from another.

## Boolean Operations

vcad supports three boolean operations:

| Operation | Operator | Method | Result |
|-----------|----------|--------|--------|
| Union | `+` | `.union(&b)` | Combines two shapes |
| Difference | `-` | `.difference(&b)` | Subtracts B from A |
| Intersection | `&` | `.intersection(&b)` | Keeps only overlap |

## Creating a Simple Hole

To create a hole, subtract a cylinder from your part:

```rust
use vcad::{centered_cube, centered_cylinder};

let plate = centered_cube("plate", 50.0, 50.0, 5.0);
let hole = centered_cylinder("hole", 5.0, 10.0, 32);

let result = plate - hole;  // Subtracts hole from plate
```

<Callout type="info">
Make your cutting cylinder longer than your part. This ensures the hole goes all the way through, even with small floating-point errors.
</Callout>

## Operator Overloads

vcad overloads Rust operators for a clean syntax:

```rust
let union = part_a + part_b;       // or part_a.union(&part_b)
let difference = part_a - part_b;  // or part_a.difference(&part_b)
let intersection = part_a & part_b; // or part_a.intersection(&part_b)
```

You can chain operations:

```rust
let result = base + feature - hole1 - hole2;
```

## Positioning Holes

Often you need holes at specific locations. Translate the cutting cylinder before subtracting:

```rust
let plate = centered_cube("plate", 100.0, 60.0, 5.0);

// Corner hole
let hole = centered_cylinder("hole", 3.0, 10.0, 32)
    .translate(40.0, 20.0, 0.0);

let result = plate - hole;
```

<Callout type="tip" title="Multiple holes">
You can union multiple holes together, then subtract them all at once. This is more efficient than subtracting one at a time.
</Callout>

## Combining Multiple Holes

For multiple holes, combine them first:

```rust
let plate = centered_cube("plate", 100.0, 60.0, 5.0);
let hole = centered_cylinder("hole", 3.0, 10.0, 32);

// Create four holes
let hole1 = hole.translate(40.0, 20.0, 0.0);
let hole2 = hole.translate(-40.0, 20.0, 0.0);
let hole3 = hole.translate(40.0, -20.0, 0.0);
let hole4 = hole.translate(-40.0, -20.0, 0.0);

let all_holes = hole1 + hole2 + hole3 + hole4;
let result = plate - all_holes;
```

## Segment Count

The `segments` parameter controls how smooth cylinders appear:

- **16**: Coarse, visible facets (fast)
- **32**: Good for most holes
- **64**: Smooth curves
- **128+**: Very smooth (slower)

```rust
// Low-poly hole (for prototyping)
let quick_hole = centered_cylinder("hole", 5.0, 10.0, 16);

// Smooth hole (for final export)
let smooth_hole = centered_cylinder("hole", 5.0, 10.0, 64);
```

## Common Patterns

### Counterbore Hole

vcad provides a helper for counterbore holes:

```rust
use vcad::counterbore_hole;

// M5 socket head cap screw counterbore
let cbore = counterbore_hole(
    5.5,   // through hole diameter
    10.0,  // counterbore diameter
    5.0,   // counterbore depth
    20.0,  // total depth
    32     // segments
);
```

### Countersink Hole

Use a cone for countersinks:

```rust
let hole = centered_cylinder("hole", 3.0, 15.0, 32);
let sink = Part::cone("sink", 6.0, 3.0, 3.0, 32)
    .translate(0.0, 0.0, -1.5);

let countersink = hole + sink;
```
